# Botanic-Vision 🌾

Botanic-Vision is a machine learning model designed to predict various diseases in rice plants with an impressive accuracy of **97.8%**. This tool is crucial for farmers and agricultural professionals to quickly identify and manage rice plant diseases, helping to ensure healthy crops and optimize yields.

### 🌟 Key Features

- **High Accuracy**: Achieved an accuracy rate of **97.8%** in predicting rice plant diseases.
- **User-Friendly Interface**: Simple and intuitive UI made using Flask, suitable for users with any level of technical expertise.
- **Disease Detection**: Identifies the following rice plant diseases:
  - Bacterial Leaf Blight
  - Leaf Smut
  - Brownspot
  - Leaf Blast

**🧠 How It Works**
